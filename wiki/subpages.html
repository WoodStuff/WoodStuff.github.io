Games

  • Point Game - A clicker game I tried to make a few months ago. Has spaghetti code, and I plan to rewrite it later.
  • AGP - A RPG game that started out as a discord server (somehow) a few years ago, and I'm somehow after all those years trying to keep it alive. You can fight enemies, and they give you currency and XP when beaten. Main article here.
  • Rocks - An idle game that has literally nothing done yet. The main currency is supposed to be rocks which have a per second and max amount.
  • Button Game, Text Game - Very minor games that have pretty similar concepts. Button game is sort of a puzzle game where you need to click buttons in the right order, and text game is also a puzzle game where you need to type what the inputs say on the placeholder. As you can probably guess, nothing really has been made right now.
  • The Thing - Similar to AGP, but inspired by a minecraft game on a friend's server. Adventure/RPG game where there are "chapters" and you can go on maps that have groups of enemies in them. Similar to Miitopia if you know what that is.
  • Coin Fields - An idle game. The main currency is Coins, which you can use to buy boxes. Then in the boxes you can buy powers like generating coins. There are multiple biomes.
  • Geomy.io - Might remove the .io later on. Basically agar.io and diep.io combined, with the gameplay of agar.io and custom tiers of diep.io. Currently in the phase of making multiplayer, which is possibly never gonna happen.
